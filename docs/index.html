To enable WebGPU for GPU-accelerated inference:

  • Chrome / Edge: Navigate to chrome://flags, search for WebGPU, set to Enabled, and relaunch.
  • Safari 18+: Go to Settings → Feature Flags (or Develop → Feature Flags), enable WebGPU.
  • Firefox Nightly: Navigate to about:config, set dom.webgpu.enabled to true, and restart.
